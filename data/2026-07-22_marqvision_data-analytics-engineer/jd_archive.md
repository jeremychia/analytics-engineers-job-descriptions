Data Analytics Engineer Job Description

Position: Data Analytics Engineer
Location: Seoul, Korea

- Experience with Airflow or similar workflow orchestration tools
- Deep understanding of SQL (PostgreSQL, Snowflake, BigQuery, etc.)
- Data warehouse layer design experience
- Query optimization in large-scale database environments
- Experience resolving data consistency issues across sources

Preferred Qualifications:
- Snowflake and dbt modeling experience
- BI platform collaboration and operations
- Unstructured data handling
- CI/CD and Git proficiency
