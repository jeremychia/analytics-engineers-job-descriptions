Analytics Engineer at Jacobs, Dublin. Impact: working on BusConnects, a major infrastructure programme improving commutes and sustainability. Greenfield opportunity as team is still maturing; help shape processes and influence how analytics is done. Day to Day: frame ambiguous business problems into structured data analysis questions and testable hypotheses; apply descriptive, inferential, and predictive analysis; perform statistical analysis using regression, hypothesis testing, sampling, confidence intervals; conduct exploratory data analysis; build reproducible analytical workflows using Python and notebooks; use AI tools to enhance data analysis, code generation, and insight discovery with careful validation; query, validate, manipulate data using SQL; create data visualisations and insight-led narratives; communicate findings supporting decision-making; collaborate with stakeholders; assess data quality, bias, limitations; work within data governance, privacy, ethical frameworks. Tech Stack: Python (pandas, numpy, scipy, statsmodels); SQL; Jupyter notebooks; Azure (Fabric, Synapse, Databricks); Azure Data Factory; AI and copilot tools; matplotlib, seaborn, Plotly; Git, CI/CD. Required: strong SQL, Python, Microsoft Fabric/Synapse/Databricks, exposure to AI, communication skills, data quality knowledge, data lineage, version control, CI/CD familiarity, cloud (Azure), leadership/mentoring.
