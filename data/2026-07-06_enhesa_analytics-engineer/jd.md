# Analytics Engineer at Enhesa - Lisbon, Portugal

Job Title: Analytics Engineer
Company: Enhesa
Location: Lisboa, Lisbon, Portugal
Seniority Level: Mid-Senior level
Employment Type: Full-time
Job Function: Information Technology
Industry: Information Services

Company Overview
Enhesa describes itself as "the leading provider of regulatory and sustainability intelligence worldwide." The organization supports global businesses in understanding environmental, health, and safety (EHS) compliance requirements across different regions and industries.

Core Responsibilities
The role focuses on developing the analytical data layer within Microsoft Fabric. Key duties include:
- Owning transformations from Silver-to-Gold layers, defining requirements and delivering production datasets
- Creating SQL and PySpark transformations with incremental processing and recovery patterns
- Designing dimensional models using fact tables, dimensions, and slowly changing dimensions
- Optimizing schemas for Power BI semantic models and self-service analytics
- Implementing quality controls through validation checks, automated testing, and monitoring
- Collaborating with Data Engineers and BI Engineers on data architecture
- Applying engineering practices: Git workflows, CI/CD, environment promotions, and access governance
- Managing stakeholder expectations regarding risks, scope, and timelines

Required Qualifications
Education: Bachelor's degree in Engineering, Computer Science, IT, or equivalent experience
Experience: 3+ years delivering production analytical data models in Analytics Engineering, Data Engineering, or Business Intelligence roles

Core Technical Skills:
- Advanced SQL (CTEs, window functions, performance tuning)
- Dimensional modeling (star schemas, SCD Type 1/2, conformed dimensions)
- Spark & Delta Lake (joins, partitioning, MERGE, OPTIMIZE, ZORDER)
- Power BI semantic layer understanding
- Data quality and observability practices
- Data access governance and least-privilege patterns

Soft Skills:
- Communication of complex data concepts to non-technical audiences
- Stakeholder partnership and negotiation
- Problem-solving and root-cause analysis
- English fluency

Nice-to-Have Skills
- dbt or equivalent transformation frameworks
- Orchestration tools (Fabric, Azure Data Factory)
- Git and CI/CD workflows
- Microsoft Fabric-specific optimization (VORDER)
- Python scripting
