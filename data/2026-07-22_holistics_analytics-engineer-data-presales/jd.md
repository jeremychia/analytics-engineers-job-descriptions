Analytics Engineer — Data & Presales (Junior to Mid-level)

Location: Ho Chi Minh City, Vietnam

Company Description
Holistics is a Business Intelligence platform built for the agentic AI era. The organization develops the BI tool Holistics, the database diagram tool dbdiagram.io, and maintains the open-source project dbml.org. Their mission centers on helping data teams build a trusted, supervised self-service analytics experience.

Core Responsibilities

Internal Data & Analytics Engineering:
- Contribute to our internal data pipeline to maintain ingestion, transformation, and reporting for stakeholders
- Build best-practice data models using dbt with dimensional modeling approaches
- Keep data trustworthy by adding tests and validation practices
- Build with coding agents & dogfood using AI tools like Development Copilot and Claude Code/Cursor to develop models on the analytics-as-code layer

Presales & Solutions (Customer-Facing):
- Craft killer demos using public datasets modeled in Holistics' platform
- Run technical POCs working directly with prospect data on schema reviews and modeling
- Support RFPs & technical Q&A paired with Sales on deep technical questions
- Partner with the sales reps joining solutioning calls to own technical substance
- Build enablement assets including demo warehouses, repos, guides, and videos

Key Requirements

Mindset & Skills:
- Genuine interest in data analytics / BI and ownership of complete data stack
- Customer-facing energy energized by discovery calls, demos, and live problem-solving
- Good communication in English (written and spoken)
- Solid SQL, data modeling familiarity, Python/equivalent scripting, Git basics, cloud data warehouse exposure

Experience level: Junior-to-mid position; prior customer-facing technical or consulting work is advantageous but not required.

Work Arrangement
Full-time, Ho Chi Minh City-based with flexible hybrid WFO/WFH setup
