BI and Analytics Engineer [up to RM12k]

Randstad (recruiting for an MNC in real estate/engineering solutions)
Location: Kuala Lumpur, Wilayah Persekutuan
Salary: RM8,000 - RM12,000 per month
Employment Type: Permanent

The role encompasses the complete BI lifecycle: requirements gathering, data modeling, ETL design, semantic layer creation, and visualization. The engineer will design interactive dashboards using Power BI with DAX and Power Query, build paginated reports via SSRS, and develop data integration workflows through SSIS. Cloud responsibilities include leveraging Microsoft Azure services to modernize infrastructure. Additional duties involve writing optimized SQL queries, identifying performance bottlenecks, and serving as a data partner to answer analytical questions and extract business insights.

Requirements

Candidates need proven BI development experience delivering end-to-end data projects. Essential skills include expert-level Power BI proficiency with data modeling expertise, deep SQL Server BI stack knowledge (SSIS/SSRS), advanced SQL optimization abilities, and practical Azure cloud experience. Strong analytical problem-solving capabilities are critical.

Nice-to-Have Qualifications

Experience migrating legacy SSIS/SSRS workloads to Azure native services and knowledge of Python or R for statistical analysis would strengthen applications.
