# Data and Analytics Engineer - SLG Job Posting

Company: Flibco
Job Title: Data and Analytics Engineer
Location: Bascharage, Luxembourg
Employment Type: Full-Time, Permanent Contract
Language Requirement: English (C1 Advanced - Read, Write, Speak)

Key Responsibilities
The role focuses on designing and building Flibco's data platform. Primary responsibilities include:
- Data Transformation: Develop and maintain dbt models across staging, intermediate, and mart layers with emphasis on "testing, documentation, and CI/CD best practices"
- Cloud Infrastructure: Optimize Snowflake environments for performance and cost, manage data governance and semantic layers
- Data Pipeline Management: Work with Fivetran integrations, manage schema updates, and monitor data reliability
- Analytics Support: Build documented data models, maintain metric consistency, and support BI tools like Tableau, Sigma, and PowerBI

Required Skills
- Strong Snowflake experience
- Advanced SQL and data modelling expertise
- Hands-on dbt or similar transformation framework experience
- Production-grade analytics platform background

Desirable Skills
- Python for automation and tooling
- CI/CD pipelines and Git workflows
- Data governance and metadata management
- AWS or Azure cloud platform experience

Company Culture
Flibco emphasizes "autonomy" and "collaborative, impact-driven culture where everyone's voice is heard." The organization focuses on "business intelligence methodologies" to improve customer satisfaction in airport transportation services.
