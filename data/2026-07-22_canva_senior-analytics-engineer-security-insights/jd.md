Senior Analytics Engineer - Security Insights

Company: Canva
Location: Sydney, Australia (based role, hybrid arrangement)
Employment Type: Full-time / Permanent

Key Responsibilities
The role focuses on:
- Powering real-time threat detection by building resilient, high-throughput pipelines for security log streams
- Building AI-ready data infrastructure on the semantic layer using Snowflake
- Enabling security teams to make informed decisions with data
- Engineering at scale handling petabyte-scale security data
- Championing reliability and efficiency through comprehensive monitoring
- Building leverage, including with AI, for self-service capabilities

Requirements
Candidates should have:
- Curiosity about security and cross-functional collaboration skills
- Knowledge how to build fault-tolerant, real-time and batch pipelines using dbt, Kafka, and streaming frameworks
- Strong SQL skills, solid Python experience with software-building capabilities
- Deep understanding of privacy, governance, and least privilege principles
- Care deeply about data reliability, cost efficiency, and observability

Benefits
Offerings include equity packages, Inclusive parental leave policy, annual Vibe & Thrive allowance, and flexible leave options.
