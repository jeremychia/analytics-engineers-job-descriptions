# Data Analytics Engineer - EVA

**Company:** EVA - Esports Virtual Arenas
**Job Title:** Data Analytics Engineer
**Location:** Paris, Île-de-France, France
**Salary:** Not stated

## Key Responsibilities

The role involves evolving an existing data stack (BigQuery, dbt, Lightdash) and democratizing its usage across teams. Primary duties include:

- Managing the current data infrastructure and integrating new data sources
- Developing analytics use cases collaboratively with business teams
- Creating and enhancing dashboards using Lightdash
- Maintaining and advancing data infrastructure through GitHub Actions and Pulumi
- Paving the way for ML/IA applications

## Requirements

**Education:** Engineering, data science, IT, mathematics, or statistics degree
**Experience:** Minimum 5 years in data engineering or analytics with technical background
**Key Skills:** Advanced SQL proficiency; familiarity with modern data stack tools like Airbyte, dbt, Snowflake, BigQuery; cloud environment knowledge; web API interaction capabilities; business-oriented mindset focused on ROI; leadership capabilities for future team building

## Work Arrangement

Direct reporting to the Chief Operating Officer; full-time position as the organization's first dedicated data team member.
