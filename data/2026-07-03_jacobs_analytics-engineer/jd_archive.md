Analytics Engineer

At Jacobs, we're challenging today to reinvent tomorrow by solving the world's most critical problems for thriving cities, resilient environments, mission-critical outcomes, operational advancement, scientific discovery and cutting-edge manufacturing, turning abstract ideas into realities that transform the world for good.

Your impact
This is a great time to join Jacobs' Dublin team as we scale our analytics capability within transportation. You'd be working on BusConnects, a major infrastructure programme that will directly improve daily commutes and make the city more sustainable. It's a real greenfield opportunity as the team is still maturing, so you'll help shape processes, influence how analytics is done, and have a tangible impact from day one. There's strong growth in the pipeline, with leadership heavily investing in this space.

Day to Day
• Frame ambiguous business problems into structured data analysis questions and testable hypotheses
• Apply core Analytics techniques including descriptive, inferential, and predictive analysis
• Perform statistical analysis using regression, hypothesis testing, sampling, and confidence intervals
• Conduct exploratory data analysis to identify patterns, trends, and anomalies in complex datasets
• Build reproducible analytical workflows using Python and notebook environments
• Use AI tools to enhance data analysis, code generation, and insight discovery with careful validation
• Query, validate, and manipulate data using SQL across multiple data sources
• Create clear data visualisations and insight-led narratives tailored to different audiences
• Communicate analytical findings in a way that supports decision-making without overstating certainty
• Collaborate with stakeholders to refine data requirements and ensure outputs are relevant and accurate
• Assess data quality, bias, and limitations, and clearly articulate what the data can and cannot support
• Work within data governance, privacy, and ethical frameworks at all times

Technical Stack
• Python for data analysis including pandas, numpy, scipy, statsmodels
• SQL for data extraction, transformation, and validation
• Jupyter or similar notebook tools for reproducible data analysis
• Azure-based data platforms including Fabric, Synapse, and Databricks
• Data pipeline tooling such as Azure Data Factory
• AI and copilot tools to support and accelerate analytical workflows
• Data visualisation using matplotlib, seaborn, Plotly or similar
• Version control and CI/CD practices using Git

Key Skills & Experience Required
• Strong experience with: SQL, Python, Microsoft Fabric, Synapse, and Databricks
• Exposure to AI and advanced analytics: Familiarity with Generative AI (AI augmentation concepts)
• Experience using analytical tools that support AI-driven insights
• Communication & Storytelling: Strong ability to communicate insights clearly
• Capable of translating data into meaningful business stories
• Data Management & Governance: Solid understanding of data quality and assurance principles
• Knowledge of data completeness and data lineage
• Engineering Best Practices: Experience with version control (e.g. Git)
• Familiarity with CI/CD pipelines and deployment practices
• Cloud Environment: Primary focus on Microsoft Azure
• Experience with AWS or Google BigQuery is transferable
• Leadership & Growth: Will be expected to support and mentor others
